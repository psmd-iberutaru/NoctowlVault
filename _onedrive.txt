The NoctowlVault is an Obsidian created Markdown-based note system.

OneDrive is used to sync the note system between the machines of 
the Pokémon Server System. OneDrive was chosen over the Nexus drive.
As such, opening and editing this note system with Obsidian should
be done from the OneDrive section.

As a precaution, these markdown notes are also transferred to the Nexus
drive, copied from OneDrive. They are also uploaded to GitHub and from 
there made into a static site. The proper scripts gluing these together
should be found on the NOCTOWL server.
